Evoluções Web

WEB 1.0 | WEB 2.0 | WEB 3.0 | WEB 4.0

Web 1.0

Baseada em hyperlinks, o usuário só podia consumir o que via, não poderia alterar e nem adicionar algo... Escassez de vídeos, limitação das ferramentas de marketing.

Web 2.0

Melhoria no marketing para interação, espaço mais dinâmico, criação de comentários em páginas de blogs, chegada das redes sociais, tecnologia ajax e expansão do CSS.

Web 3.0

Tecnologia de automação, blockchain, nft’s, machine learning, inteligência artificial e algoritmos de recomendação de conteúdo.

Web 4.0

(previsão) IA em marketing, aprendizado de máquina e tecnologias voltadas a experiência do usuário. Tendências: Internet das Coisas e Interação Social.
